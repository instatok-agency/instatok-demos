# Instatok Agency Demos

Showcase of website demos for client validation and portfolio.

## Demos

| Demo | Persona | Aesthetic |
|------|---------|-----------|
| `demo-anna-psychologist` | Greek therapist (Παγκράτι) | Warm soft maximalism, editorial |
| `demo-vasiliou-architect-v2` | Architecture studio (Παγγαίο) | Swiss minimalism, gallery-tier |

## Tech Stack

- Astro 6
- Tailwind v4
- GSAP / CSS animations
- Generated with Claude Code

## Run locally

```bash
cd demo-{name}
npm install
npm run dev
```
